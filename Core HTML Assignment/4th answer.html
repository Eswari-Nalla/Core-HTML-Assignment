List of Tags

  1. body: Defines the body of the Document
  2. h1: Defines the heading level1
  3. p: Defines the paragraph of some text
  4. b: Defines the bold text
  5. em: Defines the emphasized text
  6. strong: Defines the important text
  7. mark: Defines the marked text
  8. sup: Defines the superscripted text
  9. sub: Defines the subscripted text
  10. small: Defines the smaller text
